Key terms and concepts used throughout the Microsoft Foundry documentation.

## A

### AIProjectClient
The primary client class in the Azure AI Projects SDK (`azure-ai-projects`). Used to authenticate and interact with Azure AI Foundry resources — creating agents, managing threads, and invoking API endpoints. Supports both Entra ID (DefaultAzureCredential) and API Key authentication.

### Agent
An AI-powered entity that can perform tasks autonomously using tools, models, and instructions. In Microsoft Foundry, agents are created via the Azure AI Projects SDK and can use tools like Code Interpreter, File Search, and Function Calling.

### Azure AI Search
A cloud search service providing keyword, vector, and semantic search capabilities. Used for hybrid document retrieval in RAG patterns.

## C

### Code Interpreter
An agent tool that allows an agent to write and execute Python code in a sandboxed environment. Used for data analysis, mathematical computations, file transformations, and generating charts or visualizations.

## E

### Entra ID
Microsoft's cloud identity platform (formerly Azure Active Directory). Used for authenticating requests to Azure AI Foundry APIs via `DefaultAzureCredential` or other Azure Identity SDK credential types. The recommended authentication method for production workloads.

## F

### FastMCP
A Python framework for building Model Context Protocol (MCP) servers. The foundry-docs MCP server is built on FastMCP 3.x.

### File Search
An agent tool that enables an agent to retrieve information from uploaded documents using vector search. Supports PDF, Word, and other file formats stored in a vector store attached to the agent.

### Function Calling
An agent tool that allows an agent to invoke developer-defined functions or external APIs. The model identifies when to call a function, returns structured arguments, and the application executes the call and returns results.

## M

### MCP (Model Context Protocol)
An open protocol that enables AI assistants to connect to external data sources and tools through a standardized interface. MCP servers expose tools, resources, and prompts.

## O

### OpenAPI
A standard specification format (versions 3.0/3.1) for describing REST APIs. Microsoft Foundry API reference documentation is generated from OpenAPI specs and rendered in the Mintlify API playground, enabling interactive "Try It" testing directly in the docs.

## R

### RAG (Retrieval Augmented Generation)
A pattern that combines information retrieval with language model generation to produce grounded, accurate responses using external knowledge sources.
