This page lists endpoints available in the **preview** API versions that have not yet graduated to the **stable (v1)** release. Use preview APIs to test upcoming features before they become generally available.

<Warning>
Preview APIs may change or be removed without notice. Do not use preview endpoints in production workloads.
</Warning>

## OpenAI v1 API

The preview adds **12 endpoints** beyond the 106 in stable.

### Audio

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/audio/speech` | Generate text-to-speech audio from input text |
| `POST` | `/audio/transcriptions` | Transcribe audio into the input language |
| `POST` | `/audio/translations` | Translate spoken audio to English text |

### Images

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/images/generations` | Generate images from a text prompt |
| `POST` | `/images/edits` | Edit an image given a prompt and mask |
| `POST` | `/images/variations` | Create variations of an existing image |

### Video generation

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/videos` | Create a new video generation job |
| `GET` | `/videos` | List video generation jobs |
| `GET` | `/videos/{video-id}` | Retrieve properties of a video generation job |
| `DELETE` | `/videos/{video-id}` | Delete a video generation job |
| `GET` | `/videos/{video-id}/content` | Retrieve a thumbnail of generated video content |
| `POST` | `/videos/{video-id}/remix` | Remix an existing generated video with a new prompt |

## Projects API

The preview adds **9 endpoints** beyond the 17 in stable.

### Evaluation runs

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/evaluations/runs:run` | Create an evaluation run |
| `POST` | `/evaluations/runs:runAgent` | Create an agent evaluation run |
| `GET` | `/evaluations/runs` | List evaluation runs |
| `GET` | `/evaluations/runs/{name}` | Get an evaluation run by name |
| `POST` | `/evaluations/runs/{name}:cancel` | Cancel an evaluation run |
| `DELETE` | `/evaluations/runs/{name}` | Delete an evaluation run |

### Red team runs

| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/redTeams/runs:run` | Create a red team run |
| `GET` | `/redTeams/runs` | List red team runs |
| `GET` | `/redTeams/runs/{name}` | Get a red team run by name |
