This page tracks significant changes to the Microsoft Foundry documentation site. For platform release notes, see the [Azure updates page](https://azure.microsoft.com/updates/).

<Update label="2026-07-07" description="Navigation expansion, hosted agent content refresh, and version updates">

## Added

- **[What is Microsoft Foundry](/what-is-microsoft-foundry/what-is-foundry)** — New overview section added to the navigation sidebar.
- **[Foundry Agent Service](/agent-service/quickstart-hosted-agent)** — 95 `agent-service/` pages (quickstarts, hosted agents, voice agents, system messages, publish agents, agent catalog, knowledge and retrieval, and tracing) added to the navigation sidebar.
- **[Foundry Models](/foundry-models/deploy-foundry-models)** — 112 `foundry-models/` pages (model catalog, router, deployment, fine-tuning, and governance) added to the navigation sidebar.

## Fixed

- **[Quickstart: Deploy your first hosted agent](/agent-service/quickstart-hosted-agent)** — Corrected extension install command from `azd ext install microsoft.foundry` to `azd ext install azure.ai.agents`; updated required versions to azd CLI 1.26.0 and extension 0.1.41-preview.
- **[Manage hosted agents](/agent-service/manage-hosted-agent)**, **[Manage hosted sessions](/agent-service/manage-hosted-sessions)**, **[Migrate hosted agents](/agent-service/migrate-hosted-agent-preview)** — Updated minimum Azure Developer CLI prerequisite from version 1.23.0 to 1.26.0.

## Changed

- **[Quickstart: Deploy your first hosted agent (legacy)](/agents/development/quickstart-hosted-agent)**, **[Deploy a hosted agent (legacy)](/agents/development/deploy-hosted-agent)**, **[Hosted agents (legacy)](/agents/development/hosted-agents)**, **[Manage hosted agent lifecycle (legacy)](/agents/development/manage-hosted-agent)** — Added deprecation warnings: the initial public preview hosting backend was retired on May 22, 2026. Pages now redirect to current `agent-service/` equivalents and the migration guide.

</Update>

<Update label="March 2026" description="AI-native features and developer experience improvements">

## Added

- **Contextual menu** on all pages with options to copy as Markdown, view as Markdown, and open in AI tools (ChatGPT, Claude, Perplexity, Grok, GitHub Copilot)
- **MCP server connections** for VS Code and VS Code Insiders directly from the contextual menu
- **[Connect AI tools](/get-started/connect-ai-tools)** guide with step-by-step instructions for Claude Desktop, ChatGPT, GitHub Copilot, Cursor, Perplexity, and Grok
- **[AI ingestion formats](/get-started/ai-ingestion)** guide explaining Markdown pages, llms.txt, llms-full.txt, skill.md, and MCP server
- **AI assistant configuration** with Microsoft Foundry-specific persona, terminology, and product context
- **Documentation agent rules** (AGENTS.md) for consistent automated documentation contributions
- **Page feedback** with thumbs up/down rating, edit suggestions, and issue creation
- **This changelog** for tracking documentation updates

## Improved

- Documentation site now serves all pages as Markdown to AI agents for reduced token usage
- MCP server provides live search across all indexed documentation pages
- skill.md provides structured product context for AI agent skill registration

</Update>
